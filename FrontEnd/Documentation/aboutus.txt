# SolEspaña About Us Documentation

## Overview
The About Us page (Aboutus.html) introduces the team behind SolEspaña through a visually appealing and personal presentation of team members.

## File Structure
- Aboutus.html: Main HTML structure
- aboutus.css: Stylesheet for About Us page
- Images in "About us/" directory

## Components

### 1. Navigation Bar
- Fixed position header
- Semi-transparent with blur effect
- Links to main sections:
  - Homepage (SolEspaña)
  - Destinos
  - Consejos
  - Entre Bastidores (current page)

### 2. Hero Section
- Background image: "About us/Group 4.jpg"
- Title: "Entre Bastidores"
- Subtitle: "Conoce al equipo que hace la magia posible"

### 3. Team Member Profiles
Each team member section includes:
- Circular profile photo (150px × 150px)

- Detailed information

### 1. Layout

3. Performance Considerations
